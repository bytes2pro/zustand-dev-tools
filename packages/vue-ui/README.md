# @bytes2pro/vue-ui

Vue 3 component library template.

## Install

```bash
pnpm add @bytes2pro/vue-ui
```

## Usage

```ts
import { VButton } from '@bytes2pro/vue-ui';
app.component('VButton', VButton);
```

## Development

- Build: `pnpm build`
- Test: `pnpm test`
- Lint: `pnpm lint`
