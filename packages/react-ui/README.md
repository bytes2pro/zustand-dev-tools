# @rte/react-ui

React component library template.

## Install

```bash
pnpm add @rte/react-ui
```

## Usage

```tsx
import { Button } from '@rte/react-ui';
```

## Development

- Build: `pnpm build`
- Test: `pnpm test`
- Lint: `pnpm lint`

## Publishing

Use Changesets at repo root. See `../../docs/publishing.md`.
