# @rte/solid-ui

SolidJS component library template.

## Install

```bash
pnpm add @rte/solid-ui
```

## Usage

```tsx
import { SButton } from '@rte/solid-ui';
```

## Development

- Build: `pnpm build`
- Test: `pnpm test`
- Lint: `pnpm lint`

## Publishing

Use Changesets at repo root. See `../../docs/publishing.md`.
